    Projects

    SalesPulse DataSet

    SalesPulse is a data analytics dashboard I built in Power BI using a dataset of over 50,000 online retail transactions that I found on Kaggles Website. It goes over the customer journey from product view to checkout using SQL and Excel to show their stages. Most importantly it shows key performance indicators like AOV, CLV, churn rate, while their trends are visualized with DAX-powered metrics. The result is a comprehensive, actionable dashboard for business decision-making.

    I’ve always been interested in the data field, and I really wanted to get more comfortable using industry standard tools like Power BI and SQL. Building this project gave me a hands-on way to understand how businesses track user behavior and it really is eye opening. It lets me explore real-world metrics like conversion rates, drop-offs, and how valueable a customer is. This project really helped me look at things from a different POV.

    Smart Pantry

    Smart Pantry is a full-stack app that a couple of my friends and I helped create. It helps users manage groceries, from pantry items to ones in the fridge. It uses Firebase for secure user authentication and data storage. We used the OpenAI API, so it provides personal recipe suggestions based on what users already have.

    This idea sprung up after I took a trip to Nigeria. I was just talking to them how crazy the food waste in America is when some people are really struggling. That conversation sparked the idea to build something that could actually help reduce waste in our own home first. We wanted a solution that made managing pantry and fridge items easier while taking out the hard part... figuring out what to eat. Smart Pantry became our way of turning that conversation into action.

    Desktop Cleaner

    Desktop Cleaner is a Python automation tool I designed to organize files. It uses Python code and perform a scan to see when the last time that specific file was opened. It aviods important system files and scans to detect and categorize files older than 30, 60, and finally 90 days. This reduces visual clutter and boosts productivity by saving time spent on manual file management. This tool significantly cuts down on storage usage by removing or archiving unnecessary files.

    My desktop just had unnecessary games and files, and I got tired of manually organizing them. I figured if I could build a tool to do it for me, it would save time and keep things clean without the hassle. I also thought it would be good real-world experience as a simple convenience project but became a really helpful automation tool.
